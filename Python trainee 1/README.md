## How to start

1. Clone project from github
    ```
    git clone https://github.com/xistadi/homework
    ```
2. Create `venv`
    ```zsh
    #Linux/Mac
    python3 -m venv env
    ```
    ```bash
    #Win cmd
    python -m venv env
    ```
3. Activate `venv`
4. Install requirements
    ```
    pip install -r requirements.txt
    ```
5. Run `homework.py`
    ```zsh
    #Linux/Mac
    python3 homework.py
    ```
    ```bash
    #Win cmd
    python homework.py
    ```
